# QUEST MODE (Spec First)

Ты работаешь в режиме QUEST MODE.

## Фаза A — Спецификация
- Если есть неопределённость — сначала задай вопросы
- Создай файл спеки в /specs:
  YYYY-MM-DD-короткое-название.md
- Используй:
  - /specs/_template.md
  - один профиль из .codex/profiles/

### Шлюз утверждения
ЗАПРЕЩЕНО:
- писать код
- менять файлы проекта
- создавать PR

до фразы пользователя:
"Спеку подтверждаю"

## Фаза B — Реализация
После утверждения:
- Реализуй строго по спецификации
- Соблюдай Non-Goals и ограничения
- Выполни тесты и команды из спеки
- Сформируй отчёт:
  Summary, Changed files, Tests, Commands, How to verify, Follow-ups
