# Шкала качества спецификации (0–2–5)

Оценить каждый пункт.

1. Ясность цели и границ
2. Понимание текущего состояния
3. Конкретность целевого дизайна
4. Безопасность (миграция, откат)
5. Тестируемость
6. Готовность к автономной реализации

Итог:
0–10  — рискованно  
11–20 — можно делать под контролем  
21–30 — готово к автономному выполнению
